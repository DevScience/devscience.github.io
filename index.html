Our Mission

DevScience is a nonprofit organization that teaches computer science classes to students from 5th to 8th grade. All profits that we earn are donated back to the Fremont Unified School District, for the benefit of your students. Led by students, for students.

Classes

Scratch: Foundations of Computer Science

Scratch is a software developed by MIT Media Lab specifically designed to teach young students programming. Scratch comes with a user-friendly drag-and-drop interface that allows young students to learn the fundamentals of computer science, without the hassle of an overly-complex language or code.

Introduction to Java: Object-Oriented Programming

Java is a universally recognized programming language used by millions of developers and companies across the globe, The Java class caters towards students who already have basic programming experience, or for older students ready to tackle a programming language.
